***VR 아바타 체화감(Embodiment) 실험 시스템***

**프로젝트 개요**

본 저장소는 가상현실(VR) 환경에서 아바타 외형과 움직임 정확도가 사용자의
체화감(Sense of Embodiment)에 미치는 영향을 분석하기 위해 개발된
VR 사용자 실험 시스템의 구현 코드이다.
본 시스템은 석사학위 논문 연구의 일환으로 개발되었으며, 아바타 조건을 정밀하게 통제할 수 있는 실험 환경을 제공하는 것을 목표로 한다.


**연구 배경 및 목적**

체화감은 VR 환경에서 몰입감과 사용자 경험을 결정하는 핵심 요소이다.
기존 연구들은 주로 최적의 아바타 조건을 비교하는 데 집중해 왔으나,아바타 외형과 움직임 정확도의 조합, 그리고 작업(Task) 맥락에 따른 차이를 체계적으로 분석한 연구는 상대적으로 부족하다.
본 프로젝트의 목적은 다음과 같다.
* 아바타 외형 수준에 따른 체화감 차이 분석
* 움직임 정확도가 체화감에 미치는 영향 분석
* 작업 유형에 따라 체화감에 영향을 미치는 요인 비교


**실험 디자인**

*독립 변수*
아바타 외형
* 스틱 아바타 (낮은 현실감)
* 로봇 아바타 (중간 현실감)
* 사람 형태 아바타 (높은 현실감)

움직임 정확도
* Upper Body IK (HMD + 컨트롤러 기반 상체 IK)
* Full Body IK (3점 센서 기반 전신 IK)
* Motion Capture (6점 센서 기반 모션 캡처)

작업(Task)
* 상체 중심 작업 (버튼 누르기)
* 전신 이동 작업 (장애물 피해서 걷기)
  
본 실험은 3 × 3 반복측정 설계로 구성되었다.


**시스템 구현**

*개발 환경*
* Engine: Unity
* VR 장비: HTC Vive Pro
* 입력 장치: Vive Controller, Vive Tracker
* IK 시스템: Final IK


*주요 구현 특징*
* 아바타 외형과 제어 로직 분리 구조
* 외형 변경 시에도 동일한 제어 로직 사용 가능
* 움직임 정확도 조건의 런타임 전환
* IK / Motion Capture 조건 간 직접 비교 가능
* 거울 기반 자기 인식 환경
* 체화감 유도를 위한 실험 장치
* 실험 데이터 자동 기록
* 조건별 체화감 점수 및 실험 로그 저장


*코드 구조*
* AvatarController : 아바타 상태 및 움직임 정확도 조건 관리
* TrackingModeSwitcher : IK / Motion Capture 조건 전환
* TaskController : 버튼 누르기, 장애물 회피 등 작업 로직 처리


*체화감 측정 요소*
* 소유감 (Ownership)
* 행위감 (Agency)
* 위치감 (Self-location)
7점 Likert 척도 설문
Repeated Measures ANOVA 및 사후 분석 수행


**연구 기여**

* VR 아바타 외형과 움직임 정확도의 영향을 작업 맥락별로 분석
* 중간 현실감(로봇) 아바타의 체화감 특성 검증
* VR 아바타 설계를 위한 실험 기반 가이드라인 제시
